class flaskext.zodb.ZODB(app=None)[source]¶

ZODB extension for Flask: persistence of native Python objects.

Basic setup:

from ZODB.FileStorage import FileStorage
from flask import Flask, redirect, render_template_string
from flaskext.zodb import ZODB

ZODB_STORAGE = lambda: FileStorage('app.fs')

app = Flask(__name__)
app.config.from_object(__name__)
db = ZODB(app)

@app.route('/')
@app.route('/<message>')
def index(message=None):
    if message is not None:
        db['message'] = message
        return redirect('/')
    return render_template_string('Latest message: {{ message }}',
                                  message=db['message'])

During requests the db object acts like a Python dict. Any changes made to it directly will persist, changes made to mutable objects within will have to be marked as mutated. This is done for you if you inherit Model for your own classes and use List and Mapping as substitutes for Python’s list and dict.

Outside of requests, the object can be used as a context manager if called, yielding the root object to be used inside the context. See __call__().

class flaskext.zodb.Model(**kwargs)[source]¶

Convinience model base.

You can subclass persistent.Persistent directly if you prefer, but this base provides some conviniences.

Set attributes in instantiation:

>>> Model(title='Hello!')
Model(title='Hello!')
>> Model(title='Hello!').title
'Hello!'

Declare mutable and dynamic attributes in the class definition:

class Post(Model):
    id = UUID4
    posted_on = Timestamp
    comments = List

>>> Post()
Post(id=UUID('c3f043a8-8f1f-4381-89b3-fd1f35265925'),
     posted_on=datetime.datetime(2010, 10, 20, 15, 42, 34, 138015),
     comments=[])
>>> type(Post().comments)
<class 'persistent.list.PersistentList'>
class flaskext.zodb.Factory(callable, *args, **kwargs)[source]¶

Set a Model attribute with a callable on instantiation. Useful for delaying initiation of mutable or dynamic objects.

class Dice(Model):
    side = Factory(random.randint, 1, 6)

>>> Dice()
Dice(side=3)
>>> Dice()
Dice(side=5)
